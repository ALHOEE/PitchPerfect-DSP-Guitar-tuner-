# PitchPerfect-DSP-Guitar-tuner

A professional, real-time guitar and instrument tuner application leveraging Digital Signal Processing (DSP) for high-accuracy pitch detection. Features both a modern vector GUI and a lightweight CLI rendering mode.

## 🚀 Features
* **Advanced DSP Engine:** High-accuracy pitch detection algorithms designed for real-time audio streams.
* **Dual Interface:**
  * Modern, clean Vector GUI for desktop use.
  * Lightweight, fast CLI render directly in the terminal.
* **Cross-Platform:** Built with Python, making it easy to run on Windows, macOS, and Linux.
